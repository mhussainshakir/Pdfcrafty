About PDFCrafty

PDFCrafty was built on a simple idea: everyday PDF tasks like merging, splitting, and converting files shouldn't require uploading your documents to someone else's server, creating an account, or hitting an arbitrary usage limit.

Every tool on this site runs entirely inside your web browser. When you merge two PDFs or compress a file, the processing happens locally on your own device using modern JavaScript libraries — your files are never transmitted anywhere. This means PDFCrafty works just as well for a casual one-off task as it does for sensitive documents like contracts, financial statements, or personal records.

Our Principles

Privacy first. We don't see your files, and we don't want to. Processing locally in your browser isn't just a feature — it's the foundation of how every tool is built.

No friction. No account creation, no email verification, no credit card. Open a tool and start using it immediately.

Genuinely free. Every tool is free to use, with no daily limits, no watermarks added to your files, and no premium tier hiding basic features.

How It Works

PDFCrafty uses well-established open-source JavaScript libraries that run directly in your browser to read, modify, and generate PDF files. This is the same technology that powers PDF viewing in browsers like Firefox, applied here to give you editing capabilities without a server in the loop.
